## O QUE É ESTE BOT?

PRIMEIRA FASE CONCLUIDA:

O Bot de Gestão Financeira é o assistente automatizado da Farm Home para gerenciar documentos fiscais. Ele:

- Recebe fotos de boletos, pedidos e notas fiscais
- Lê automaticamente as informações usando IA (Google Gemini)
- Salva tudo organizado no Supabase
- Alerta sobre vencimentos próximos
- Gera relatórios quando solicitado

---

SEGUNDA FASE SENDO INICIADA:

Apos essa primeira fase, iremos criar uma continuação da ferramenta.

Nessa fase o objetivo é criar um web app, para ser a interface e painel de trabalho e visualização dos dados extraidos, e tambem iremos criar algumas funçóes extras, para dar a sequencia e ter o resultado final esperado.

Funções esperadas:
- painel com todos os documentos a vencer, em estilo planilha, com colunas de data, fornecedor, valor, status, etc.
- listas de pagamentos por semana (de sabádo a sexta),  
- graficos interativos.
- possibilidade de inserir manualmente os dados de um documento, caso a IA não consiga extrair corretamente.
- função de importar, ler extrato bancário, para conciliação automática,
- função de inserir arquivo com os pagamentos do dda, para conferencia e ajuste dos pagamentos.
- ter uma função extremamente inteligente, integrada com IA, que permita, criar estratégias gerenciais para atrasar, renegociar, avaliar os riscos de cada documento, e sugerir as melhores ações a serem tomadas.
- poder gerar relatórios em pdf, com a possibilidade de customizar o relatorio, escolhendo as colunas, os filtros, o periodo, etc.
